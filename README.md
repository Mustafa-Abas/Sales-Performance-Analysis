# 📊Sales Performance & Business Insights Dashboard

## 📌Project Overview
This project presents an end-to-end Business Intelligence solution built using Power BI to analyze sales performance, profitability, customer behavior, and pricing strategies. The goal is to transform raw data into actionable insights that support data-driven decision-making

## 🎯 Objectives
Analyze sales and profit performance across time, categories, and regions
Evaluate discount effectiveness and its impact on revenue
Segment customers using RFM analysis to improve retention strategies
Identify root causes of underperformance
Build forecasting models to predict future sales trends

## 📊 Dashboard Features
KPI tracking (Sales, Profit, Profit Margin, Orders)
Time-series analysis and trend visualization
Customer segmentation (RFM: Champions, At Risk, Lost)
Root Cause Analysis using decomposition techniques
What-if analysis for scenario simulation
Forecasting model for future performance prediction

## 🔍 Key Insights
Low discount efficiency (0.43) indicates overuse of discounts with limited financial impact
High percentage of lost customers highlights retention challenges
Sales show steady growth, but profitability requires optimization
Technology category is the top performer in both sales and profit

## 💡 Business Impact

The analysis provides actionable recommendations to optimize pricing strategies, improve customer retention, and focus on high-performing segments to drive sustainable business growth.
